# Judges

Image judges:
- identity similarity
- prompt adherence
- quality
- diversity
- safety

Video judges:
- temporal identity consistency
- clip stability
- flicker

Audio judges:
- loudness compliance
- clipping detection

Outputs:
- `outputs/eval/<run_id>/leaderboard.csv`
- `outputs/eval/<run_id>/scores.json`
- `outputs/eval/<run_id>/project_scores.json`
- `outputs/eval/<run_id>/scene_scores.json`
